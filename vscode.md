# VS Code

## Tips & Tricks

### Enable comments in JSON files
- Go to Settings
- add a files.associations by clicking "Add Item", add `*.json` for the item and `jsonc` for the value OR just add to JSON format:
```json
"files.associations": {
    "*.json": "jsonc"
}
```
